# 🌉 Skill: Bridge
Enables seamless context sharing between multiple Antigravity projects.

## Capabilities
- **Cross-Link:** Automatically discover and link sibling projects in `context/links.md`.
- **Handoff:** Generate state snapshots for model-to-model transitions.
- **Sync:** Basic telemetry-less sync for shared rule configurations.

## Usage
Used when `bridge` or `handoff` keywords are detected.
